# RULES

Jangan menghapus kode yang bekerja.

Jangan rewrite seluruh project.

Edit seminimal mungkin.

Gunakan JavaScript vanilla.

Tidak boleh memakai framework.

Gunakan Local Storage.

Tidak boleh memakai backend.

Semua fitur harus berjalan offline.

Pisahkan CSS dan JavaScript.

Buat kode modular.

Komentari fungsi penting.

Pastikan responsive.
